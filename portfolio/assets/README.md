# Portfolio Assets

Generated by the MedAsset pipeline. Run from repo root:

```bash
python3 scripts/generate-demo-assets.py
# or
cd pipeline && make validate
```

| Asset | Source |
|-------|--------|
| `qc-report.json` | `make qc` |
| `segmentation-metrics.json` | `make metrics` |
| `*.png` overlays | `make figures` |
| `sample-spleen.dcm` | `make export` |

Replace generated assets with your own Slicer session screenshots before survey submission for maximum authenticity.
